# Employee Time Tracking System

## Overview

This Java program consists of two classes, `Employee` and `Shift`, used to manage and track employee work shifts.

---

## Features

* Records employee details (name and ID)
* Tracks sign-in and sign-out times
* Stores multiple shifts per employee
* Retrieves shifts by date or week
* Calculates total hours worked
* Represents individual shifts with start and end times

---

## Classes

### Employee

* Manages employee information and shift history
* Handles sign-in and sign-out
* Provides methods to check attendance and retrieve shifts
* Calculates total hours worked in a given week

### Shift

* Represents a single work shift
* Stores start and end date/time
* Checks if a shift occurred on a specific date or within a week
* Calculates shift duration

---

## Dependencies

* `CalendarTime`
* `Date`
* `Time`
* `Week`
* `Duration`
